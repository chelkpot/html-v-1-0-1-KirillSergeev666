Задание 5: Тег <pre>

Используйте тег <pre>, чтобы отобразить текст точно так, как он набран, сохраняя пробелы и переносы строк:

    function example() {
        let x = 10;
        let y = 20;
        
        if (x < y) {
            console.log("x меньше y");
        } else {
            console.log("x больше или равно y");
        }
        
        return x + y;
    }
